# YourGame

În cadrul acestui proiect vom dezvolta un joc web de tip shooter folosind game engine-ul Unity, compilat în format WebGL pentru a rula direct în browser. Jocul se va desfășura într-un poligon de tragere, unde jucătorul va putea interacționa cu diverse ținte și elemente specifice genului. Vor fi implementate mecanici simple și intuitive, caracteristice acestui tip de joc: deplasare, țintire, tragere și sistem de scor.

Pe lângă partea de gameplay, proiectul va include și un server web realizat în .NET 8, care va avea rolul de backend pentru joc. Serverul va oferi funcționalități precum: gestionarea scorurilor și salvarea rezultatelor jucătorilor, comunicarea dintre joc și infrastructura externă prin API-uri REST.

Astfel, proiectul îmbină partea vizuală și interactivă oferită de Unity cu logica și procesarea dinamică realizată pe serverul .NET, rezultând un sistem modern, modular și ușor de extins.

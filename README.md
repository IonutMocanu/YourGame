# YourGame

In cadrul acestui proiect o sa dezvoltam un joc web cu ajutorul game engine-ului Unity. Jocul va consta intr-un simplu shooter, toata actiunea intamplandu-se intr un poligon. Vom implementa mecanici simple, caracteristice acestui tip de joc. Proiectul va include și un server web realizat în .NET 8, care va avea rolul de backend pentru joc.
